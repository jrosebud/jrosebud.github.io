    Graphic / Photo Tools

    • Adobe Spark Post - For iOS and Android. Make simple social graphics with photos and text.
    • Google's Autodraw - If you don't have graphics folks or are a terrible artist and need simple icons, this is a great source. Scribble something and let Google try to guess what you're drawing. (You can download the resulting icons.)
    • Piktochart and Infogram - Two infographic generators. Can be embedded on a webpage or within most content platforms (like WordPress).
    • Canva - Another graphics creator.
    • Giphy Capture - Desktop app to create gifs (Mac only).
    • iFakeTextMessage - If you want to display a hypothetical text message conversation, you can create a static image using this tool.
    • Texting Story Chat Maker - If you want to simulate a text conversation and make it a video, you can use this tool. Free, with some extra features unlocked if you pay. For both iPhone and Android.

    Video Tools

    • Adobe Spark Video - Similar to Spark Post, a way to quickly make social videos with text, stills, videos, recordings, and background music.
    • Adobe Rush - Quick editing software that works on your phone, tablet, or desktop. Like a stripped-down (but still powerful) version of Adobe Premiere. Available for Mac, Windows, iOS, and Android.

    Audio Tools

    • Anchor - Mobile app that allows you to create your own radio show quickly using just a mobile phone. Record quick news briefs, allow people to respond and call in, post your clips to social media quickly. (At Annenberg Media, we use Anchor to do a 5-minute quick news update at noon every day.)
    • SoundCloud - Great place to upload any audio stories. Embeddable just about anywhere.
    • SoundCite - Tool to create audio that you can embed inline into a story. For instance, if you're writing about a protest, you can add some natural sound into your story using SoundCite.
    • Headliner - A way to create visually compelling audio clips with custom backgrounds and waveforms. Can be embedded on Twitter and Facebook, and great for audio teasers.

    Layout Tools (for standalone special projects, for example)

    • Adobe Spark Pages - A quick way to create a compelling page layout without HTML. Drag and drop! Can be embedded into a blank HTML page and is mobile friendly.
    • Storyform - A way to create responsive story layouts. More of a magazine style, where you swipe from page to page.
    • Ceros - Similar to Adobe Spark Pages; you can create a one-page scrolling story/experience. A bit more robust than Adobe Spark Pages. Sample here from Legacy High School.
    • Playbuzz - Another tool for creating slide-like, full-page stories that advance from one window to another. Embeddable in pages! Also great for creating quizzes.

    360 Tools

    • Thinglink - Create 360 experiences that can be annotated. For a student example, check out this story and scroll to the section about "Gram's Room."
    • RoundMe - Another 360 experience creator. Available as a mobile app as well.
