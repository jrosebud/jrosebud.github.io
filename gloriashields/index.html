    Social Aggregators

    • SnapWidget - A way to create photo grids from social media posts and embed them into websites.
    • Banjo - An app you can use to discover what's happening around the world for social media. Useful for global coverage, or if there's an event happening in a specific place and you want to source social media posts from that event.

    Graphic / Photo Tools

    • Adobe Spark Post - For iOS and Android. Make simple social graphics with photos and text.
    • Over - Another mobile graphic generator. Allows you to choose which platform you're going to make graphics for.
    • Google's Autodraw - If you don't have graphics folks or are a terrible artist and need simple icons, this is a great source. Scribble something and let Google try to guess what you're drawing. (You can download the resulting icons.)
    • Piktochart and Infogram - Two infographic generators. Can be embedded on a webpage or within most content platforms (like WordPress).
    • Canva - Another graphics creator.
    • Giphy Capture - Desktop app to create gifs (Mac only).
    • Diptic - Program for mobile or desktop that allows you to quickly create tiled collages.
    • Instagram's Layout app - this does the same thing as Diptic but is Instagram-specific.

    Video Tools

    • Adobe Spark Video - Similar to Spark Post, a way to quickly make social videos with text, stills, videos, recordings, and background music.
    • SoundCloud - Great place to upload any audio stories. Embeddable just about anywhere.
    • SoundCite - Tool to create audio that you can embed inline into a story. For instance, if you're writing about a protest, you can add some natural sound into your story using SoundCite.
    • Audiogram - A way to create visually compelling audio clips with custom backgrounds and waveforms. Can be embedded on Twitter and Facebook, and great for audio teasers.

    Audio Tools

    • Anchor - Mobile app that allows you to create your own radio show quickly using just a mobile phone. Record quick news briefs, allow people to respond and call in, post your clips to social media quickly. (At Annenberg Media, we use Anchor to do a 5-minute quick news update at noon every day.)
    • Periscope - For live broadcasts direct to Twitter. If you're not using Facebook Live, this is a good alternative.

    Layout Tools (for standalone special projects, for example)

    • Adobe Spark Pages - A quick way to create a compelling page layout without HTML. Drag and drop! Can be embedded into a blank HTML page and is mobile friendly.
    • Storyform - A way to create responsive story layouts. More of a magazine style, where you swipe from page to page.

    360 Tools

    • Thinglink - Create 360 experiences that can be annotated. For a student example, check out this story and scroll to the section about "Gram's Room."
    • RoundMe - Another 360 experience creator. Available as a mobile app as well.

    Other Online Tools

    • StoryMap JS - Create a curated map with clickable points. You can embed different kinds of media into the map. Great if you have a story tied to geography or various points in your town. Embeddable into a webpage or most content management systems.
    • Timeline JS - Create an interactive timeline with all sorts of media. Embeddable into a webpage or most content management systems.
    • Chatfuel - If you want to experiment with creating a chatbot on Facebook Messenger, this is a great place to start. Requires no code.
    • MailChimp - Newsletters aren't just for brands! They're a great way to keep readers informed, reach out to alumni, and just keep people up to date on your great content. If you're looking to start small and start free, MailChimp is a terrific place to start. Super simple to use and configure, and easy to embed the sign up form on pages or WordPress.
    • Medium - Need to just publish quickly and don't want to fiddle with templates or figure out how to start up a WordPress or SquareSpace site? Use Medium. Write and publish right away. It'll look a bit vanilla, but you can quickly share and distribute your content, and Medium is a robust publishing network. Great if you don't have a lot of web resources.
