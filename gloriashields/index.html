    Web Accessibility / Color Blindness Resources

    • Accessible Social Media - Fantastic tips and a checklist from accessibility advocate Alexa Heinrich on making your images and posts accessible across different social media platforms. You can also read about alt text for social media images and using icons and emojis in a more accessible fashion on her site.
    • Accessible Social Media checklist - A handy checklist to use for social media teams.
    • Color Blindness Simulator - Ever wonder what the world looks like to someone who is color-blind? Want to check that your design has good contrast and still looks fine even to someone who can't see all its colors? Use this color-blindness checker to test!
    • Color contrast checker - Is your color combination (background color + text) difficult to read? Does it meet accessibility standards? This is a web-based contrast checker you can use.
    • Pika - If you're a Mac user, this is a free utility you can use to check contrast, and the eyedropper can pick any color out of any program you have up on your screen.
    • TGPI - A contrast checker for both PC and Mac.

    Color Tools

    • Hex to CMYK converter - For when you have a hex code for a color but need a CMYK value.
    • Color Shade Generator - For when you have a base color but need darker or lighter shades of that color.
    • Adobe Color Wheel - For when you want to generate palettes based on certain color rules. Bonus: If you have an Adobe account, you can save your color palettes to your libraries for use across all Adobe products.
    • Color Picker - For when you don't have access to the color picker in Photoshop or Illustrator.

    Graphic / Photo Tools

    • remove.bg - For when you need to a photo cutout but you don't want to teach someone how to do it in Photoshop.
    • Magic Eraser - For quick photo editing, like removing unwanted objects.
    • The Noun Project - Royalty-free icons of all kinds of objects, drawn in all kinds of styles.
    • Google's Autodraw - If you don't have graphics folks or are a terrible artist and need simple icons, this is a great source. Scribble something and let Google try to guess what you're drawing. (You can download the resulting icons.)
    • iFakeTextMessage - If you want to display a hypothetical text message conversation, you can create a static image using this tool.
    • Texting Story Chat Maker - If you want to simulate a text conversation and make it a video, you can use this tool. Free, with some extra features unlocked if you pay. For both iPhone and Android.

    Instgram Examples

    • KUNC 91.5 - NPR station in Northern Colorado. Lots of examples of reels that have been created in a way that they work both as a square (the original size) but crop correctly when they get displayed as vertical rectangles for reels. Also various styles for promotional tiles (image only, with text overlay, and text-only).
    • Daily Trojan - Independent student paper from the University of Southern California. Great text-only treatments and solid, consistent graphic style.
    • WLRN - NPR station for Miami. Good consistency in text treatment and logo placement, and fun styles for themed ads (see the Zip Odes posts as an example -- there are also some great animated posts there).
